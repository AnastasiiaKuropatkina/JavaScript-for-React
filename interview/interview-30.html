== ===

== c приведением к типам: строка = number //true

=== без привидение к типам


0 == false //true
0 === false  //false
1 == "1" //true
1 === "1" //false
null == undefined //true !!!
null === undefined //false
'0' == false  //true
'0' === false //false
[] == [] //false
[] === [] //false
{} == {} //false
{} === {} //false

Object.freeze(obj) - Метод Object.freeze() замораживает объект: это значит, что он предотвращает добавление новых свойств к объекту, удаление старых свойств из объекта и изменение существующих свойств

НО! Object.freeze - видит только примитивы, изминения вложених массивах и обектах не видет

Object.isSealed(obj) - Определяет, является ли объект запечатанным (sealed).
